Senior Project Manager

Senior project manager is generally a team of individuals at the highest level of organizational management who have the day-to-day responsibilities of managing a company or corporation, they hold specific executive powers conferred onto them with and by authority of the board of directors and/or the shareholders.

Project Manager

A project manager is a professional in the field of project management. Project managers can have the responsibility of the planning, execution and closing of any project. A project manager is often a client representative and has to determine and implement the exact needs of the client.

Web Developer & Designer

A web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications, or distributed network applications that are run over HTTP from a web server to a web browser.
